Peter Answer este o aplicație interactivă care simulează un oracol virtual capabil să ofere răspunsuri aparent misterioase la întrebările utilizatorului.
Acest proiect simulează interfața și funcționalitatea celebrului site Peter Answer, cunoscut pentru iluzia sa de a "ghici" răspunsul la orice întrebare. În realitate, este un simulator care folosește un truc inteligent de tastare pentru a convinge utilizatorul că Peter deține cunoașterea absolută.
